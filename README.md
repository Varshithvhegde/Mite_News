# Mite_News
Mangalore Institute of Technology And Engineering Student connect App

# Description  
- It is College Connect Android App for Mangalore Institute of Technology And Engineering.

# Features  
- Login
- Signup
- Profile Viewing
- Adding Post(Images)
- Viewing Others Post(Images)
